# Code of Conduct

## Our Pledge
We are committed to a welcoming and harassment-free community for everyone.

## Standards
Examples of positive behavior:
- Respectful communication
- Constructive feedback
- Empathy and patience

Examples of unacceptable behavior:
- Harassment, intimidation, or abuse
- Discriminatory language
- Public or private personal attacks

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, or issues that violate this code of conduct.

## Reporting
Please report incidents to the maintainers listed in the repository contacts.
