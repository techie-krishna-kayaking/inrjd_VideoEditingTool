# Roadmap

## Near Term
- Add richer analyzer metrics (scene-change density and motion score).
- Add per-workflow benchmark snapshots in output/reports/performance.json.
- Add optional frame-interpolation profile for high-motion source clips.

## Mid Term
- Add deterministic render mode for exact reproducibility in CI.
- Add codec-specific presets for streaming and archival outputs.
- Add report dashboard generator (static HTML generated offline).

## Long Term
- Add incremental cache for reusing prepared clip transforms across runs.
- Add pluggable transition packs while preserving core architecture.
- Add reproducible packaging pipeline signatures for release artifacts.
